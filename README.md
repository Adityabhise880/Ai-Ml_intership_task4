# Ai-Ml_intership_task4
# Task 4: Feature Encoding & Scaling

## Overview
This project processes the Adult Income Dataset to prepare it for Machine Learning models.

## Steps Completed
1. **Data Cleaning:** Removed missing values.
2. **Label Encoding:** Applied to the target variable (`income`).
3. **One-Hot Encoding:** Applied to categorical features (e.g., `occupation`, `race`).
4. **Feature Scaling:** Applied `StandardScaler` to numerical features (e.g., `age`, `hours-per-week`).

## Files
* `adult_dataset.py`: Python script for processing.
* `adult_preprocessed.csv`: The final scaled and encoded dataset.
